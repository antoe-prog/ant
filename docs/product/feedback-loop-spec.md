# 피드백 루프 (7-2 산출물)

[7-2 계획](../plan/07-02-feedback-loop.md)의 **수집 위치·불만 분류·사이클**을 적는다.

<a id="feedback-loop-spec"></a>

## 수집

- 앱 내 **한 곳 이상**: 별점·자유 코멘트·버그 리포트 링크 등.  
- [6-2](log-analytics-spec.md)의 이벤트와 **ID·세션을 맞출지** 한 줄로 정한다.

## 개선 사이클

- **상위 3개 불만 유형**은 출처를 명시한다: **앱 리뷰** / **인앱 피드백** / **로그 기반 분류** 중 무엇인지.  
- **2주** 단위로 백로그에 넣고, [7-1](soft-launch-spec.md) 기간과 겹치면 우선순위를 조정한다.
