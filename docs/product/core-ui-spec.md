# 핵심 UI·인터랙션 (4-2 산출물)

[4-2 계획](../plan/04-02-core-ui.md)에서 정한 **채팅 중심 UI** 피드백을 여기에 둔다. 상세 동작은 [AI 채팅](feature-ai-chat.md)과 **같은 PR**에서 맞춘다.

<a id="core-ui-spec"></a>

## 범위 (이 템플릿)

- **핵심 화면:** **채팅**(스트리밍·에러)·**히스토리**·**설정** — 소셜 **피드**는 Core에 없으면 **문서에 넣지 않는다.**  
- **로딩:** 요청 중 스켈레톤 또는 인디케이터, 스트리밍 토큰 표시.  
- **완료·실패:** 성공 문구, **취소·재시도** 노출([2-1 V1 게이트](../product-scope.md#v1-mvp-scope)와 연계).

## 접근성·모션

- 과한 애니메이션 금지, **핵심 1~2개**만. [4-3](performance-accessibility-spec.md#performance-accessibility-spec)과 중복 시 한쪽에만 상세.
