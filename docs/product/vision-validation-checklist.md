# 비전 검증 체크리스트

`docs/product/vision-recommendations.md` 권장 5번 반영. 한 줄 비전과 [반복 시나리오](recurring-scenarios.md)를 적은 뒤 리뷰할 때 사용한다.

- [ ] **단독 이해:** 경쟁 앱 이름 없이, 한 줄만 읽어도 “누구를 위한 무엇”인지 이해할 수 있는가?
- [ ] **첫 가치 시간:** 첫 실행 후 **5분 안**에 시나리오 하나를 끝낼 수 있는가? (또는 명시한 주기 안에 첫 성공이 가능한가?)
- [ ] **타깃 공감:** 타깃 사용자에게 한 줄을 읽어 줄 때, 고개를 끄덕이거나 “그게 나다”라고 할 만한가?

전부 예이면 비전 초안 통과로 보고, 다음 단계 `01-02` 타깃 세분화로 진행한다.
