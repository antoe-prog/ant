# 컨텍스트 예산 먼저

## 우선순위

보안·SSOT·API 계약 스킬이 열리면 **절약보다 정확성**이 우선이다. ([token-skills-governance](../token-skills-governance/SKILL.md))

## 지침

1. 답하기 전 **목표 1문장 + 입력 범위 + 금지(하지 않을 것)** 를 최대 5줄로 적는다.  
2. 그 다음에만 도구/파일을 연다.  
3. 이미 충분한 정보가 있으면 **추가 Read를 하지 않는다**.
