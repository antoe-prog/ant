# API는 표·JSON 우선

## 지침

1. 필드 표(`이름|타입|필수`) 후 **한 개** JSON 예시.  
2. 장문 서술은 **예외·에러코드**에만 사용.  
3. 링크로 스키마 파일에 위임 가능하면 **링크 한 줄**로 끝낸다.
