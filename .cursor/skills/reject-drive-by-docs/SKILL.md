# 문서 drive-by 금지

## 지침

1. 사용자가 **명시한 경로**의 문서만 연다/쓴다.  
2. README·CHANGELOG는 **요청 또는 릴리스 스킬**이 있을 때만.  
3. “나중을 위해” 긴 설명 md를 **만들지 않는다**.
