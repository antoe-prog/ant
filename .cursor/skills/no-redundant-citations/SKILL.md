# 중복 인용 줄이기

## 지침

1. 이미 대화에 있는 경로는 **줄 번호만** 갱신한다.  
2. 인용은 **핵심 15줄 이내**; 나머지는 `...` 또는 “나머지 동일 패턴”.  
3. 동일 파일을 **두 번 인용하지 않는다** (새 정보만 추가).
